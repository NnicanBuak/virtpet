# virtpet
## About
Portable virtual pets
## Usage
```mermaid
---
title: Use Case
---
flowchart LR
    A[virtpet.py] -->|create instance| B1[%PET_NAME%.py\nportable script with self-saved data between sessions]
```
1. Run `virtpet.py`
2. Choose your pet and give it a name.
3. Run created `%pet_name%.py`
